# ⏰ タスクバーの時計

Windowsのタスクバーに表示される時計をモダンでスタイリッシュな表示に変更します。
時刻の前後に装飾を追加したり、曜日を可愛い絵文字で表示したり...
普段何気なく見ている時計が、デスクトップのアクセントになります。

## ✨ できること

### 🎨 スタイリッシュな時刻表示
時刻の表示を「⚡12:00⚡」のようなデザインに変更します。
シンプルながらも存在感のある表示で、タスクバーのアクセントになります。

### 📏 見やすい大きさに
- 横幅を12倍に拡大
- 高さを2.5倍に拡大
時刻が見やすくなるだけでなく、タスクバーのデザインのバランスも良くなります。

### 🎯 曜日を絵文字でパッと分かりやすく
曜日ごとに異なる絵文字を表示します：
- 日曜日は陽（🌞）
- 月曜日は月（🌙）
- 火曜日は炎（🔥）
- 水曜日は水（💧）
- 木曜日は葉（🌿）
- 金曜日は星（⭐）
- 土曜日は虹（🌈）

## 📥 インストール方法
1. まず[Windhawk](https://github.com/ramensoftware/windhawk)をインストール
2. このModを有効化
3. すぐに時計の表示が変更されます！

## 🔧 技術的な詳細
Windows APIをフックして時計の表示をカスタマイズしています：
- `GetTimeFormatEx` - 時刻の表示形式を変更
- `GetDateFormatEx` - 日付と曜日の表示を変更
- `GetWindowRect` - 時計の表示領域を取得
- `SetWindowPos` - 時計の表示サイズを変更

安全性を重視し、各APIの呼び出し時には丁寧なエラー処理を行っています。

## ⚠️ 注意点
- プライマリモニターのタスクバーでのみ動作します
- お使いのWindowsのバージョンによっては、一部の絵文字が正しく表示されない場合があります
